Your lifemap code

Copy this code and save it somewhere (e.g. email it to yourself or save it on your computer) — it contains all the information necessary to recreate your lifemap. Paste it back in this box in the future in order to modify, add to, or regenerate your lifemap.

Custom styling

The styling of your lifemap is also editable. If you change this from the default, save it as well.

Lifemapper is a fully client-side application. Your lifemap data is never transmitted anywhere. If you are concerned about privacy, you could load the page, disconnect your internet, and only then input your data. Or, you could download a local copy of the web site (by cloning the github respository) and run it all locally.
